A multithreaded version of the quicksort algorithm using a thread pool and work queue format.  Impletented using C++ on linux with p-threads.

execute using ./quicksort [number of threads to use]


The program will generate a random array of 100 ints then display the unsorted array.

The program will sort the array gives messages about how many remaining worker threads there are.

Once the sorting is done, the sorted array will be displayed.
